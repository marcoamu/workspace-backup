# Data Exporter

Exporta datos de la base de datos o archivos.

## Cuándo usarla
- Usuario necesita exportar datos
- Quiere backup en formato específico
- Necesitaレポート en PDF/CSV

## Formatos soportados
- CSV
- JSON  
- Markdown
- HTML (informes)

## Pasos
1. Consulta datos (API o DB)
2. Convierte al formato deseado
3. Guarda o muestra al usuario
