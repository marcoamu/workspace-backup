# Alert Monitor

Monitorea condiciones y alerta al usuario.

## Cuándo usarla
- Necesita notificaciones de eventos
- Quiere alertas de sistema
- Monitoring de procesos

## Tipos de alertas
- Email/Telegram
- Webhook
- Log

## Configuración
- Define condiciones a monitorear
- Frecuencia de check
- Canal de notificación
